测试 IP：`154.12.190.32`

**3. 日本 Eyeball 常规套餐 – 三网回程 CMIN**

路由是三网 CMI 回程，电信和联通去程走 CTG GIA，移动去程走 CMI。同样是 1 个 IPv4 和 1 个 IPv6 /64，流量超了会暂停。
